# 배열
## 배열의 선언
```javascript
var a = [];
var a = [1,2,3,"hello", null, true, []];
```
배열 안에는 모든 타입이 다 들어갈 수 있다. (함수, 배열, 객체 등)
new Array() 문으로 선언할 수 있지만 보통은 간단히 `[]`를 사용한다.
