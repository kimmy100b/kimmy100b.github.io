# 들어가며

vue코딩을 할 때 오른쪽 상단에 웹 브라우저 팝업창이 떠서 코드 보기가 불편했다.

# 해결방법

`Settings > Tools > Web Browsers`에 가서 체크를 해지해주면 된다.
