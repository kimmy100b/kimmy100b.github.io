# 문제상황

```
javax.servlet.jsp cannot be resolved to a type
```

# 문제확인

프로젝트를 처음 만들 때 Target Runtime을 선택하지 않아서 생긴 문제

# 문제해결

프로젝트 우클릭 > properties > Java Build Path > Libraries > Add Library... > Server Runtime > Apache Tomcat 클릭
