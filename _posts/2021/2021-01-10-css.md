```html
<textarea class="content"></textarea>
```

# <textarea> 테두리 없애기

```css
.content {
  border: none;
}
```

# <textarea> 클릭 시 테두리 없애기

```css
.content:focus {
  outline: none;
}
```

# <textarea> 크기 조절 없애기

```css
.content {
  resize: none;
}
```
