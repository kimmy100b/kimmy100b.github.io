# 변수, 상수, 리터럴

**변수(variable)란** <br>
하나의 값을 저장하기 위한 공간이다.

**상수(constant)란** <br>
값을 한번만 저장할 수 있는 공간이며 값을 변경할 수 없다.

**리터럴(literal)란** <br>
그 자체로 값을 의미하는 것

예시로 알아보자

```java
int year = 2021;  // year : 변수, 2021 : 리터럴
final int SIZE = 500;  // SIZE : 상수, 500 : 리터럴
```
